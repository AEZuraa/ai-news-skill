Собери свежие AI-новости. 
Проверь создано ли виртуальное окружение в проекте. Если нет, то создай и активируй его.
Запусти Python-скрипт из корня проекта:

```
python fetch_news.py --days 3 --limit 20 --format markdown
```

Если пользователь передал аргументы в $ARGUMENTS - учти их:
- упоминание количества дней => --days N
- количество новостей => --limit N
- "json" => --format json
- "все" / "все подряд" / "без фильтра" => --all

Покажи результат пользователю. Если новостей получилось больше 10, выдели 3-5 самых заметных наверху, а потом покажи остальные.

Для каждой статьи — без исключений, включая ArXiv — обязательно выводи три элемента:
1. Ссылка (URL)
2. Источник (название издания или сайта)
3. Одна строка о чём материал (из поля summary или заголовка)

Не группируй несколько статей в один пункт типа "ArXiv (N papers)". Каждая статья — отдельная строка.
